# QQUnrecalled (Xposed)

Prevent QQ from recalling chat messages

Only support mobileQQ, tested with v6.2.0.2655 and v6.2.1.2690

防止QQ撤回消息

只支持QQ手机版

测试过：v6.2.0.2655 和 v6.2.1.2690

因为某人“参考”（拷贝）源代码作为自己的作品 我决定就算有新版这里的源代码也不会再更新了！
